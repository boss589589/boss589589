# 國家賠償訴訟「程序不法」攻防策略說明

**報告日期：** 115年6月19日
**作者：** Manus AI

## 1. 引言

本文件旨在根據律師的專業評析，詳細闡述國家賠償訴訟中，如何將各機關公務員於執行職務行使公權力時所造成的「程序不法」行為，轉化為主張國家賠償責任的核心攻防策略。本策略將聚焦於證明這些程序違失如何不法侵害請求權人的訴訟權，進而造成財產上及非財產上之損害。

## 2. 核心法律依據：國家賠償法第2條第2項

本案主張國家賠償的核心法律依據為《國家賠償法》第2條第2項：

> 「公務員於執行職務行使公權力時，因故意或過失不法侵害人民自由或權利者，國家應負損害賠償責任。公務員怠於執行職務，致人民自由或權利遭受損害者亦同。」

本策略將證明各賠償義務機關所屬公務員的程序違失，構成對請求權人憲法第16條所保障之訴訟權的「不法侵害」或「怠於執行職務」，從而國家應負賠償責任。

## 3. 關鍵「程序不法」行為及其法律意涵

### 3.1 檢察機關（臺灣臺中地方檢察署 - 檢察官黃裕峯）之程序不法

| 違失行為 | 法律依據/義務 | 法律意涵與攻防重點 |
| :--- | :--- | :--- |
| **以相驗檢察官身分逕行偵查及不起訴** | 《刑事訴訟法》第218條（相驗義務） | 越權處分，規避偵查檢察官獨立審查，剝奪請求權人受合法偵查之權利。 |
| **傳喚程序未合法** | 《刑事訴訟法》第71條第4項（傳票應由檢察官簽名） | 傳喚程序自始無效（參最高法院96台抗152），導致請求權人參與偵查之程序權利受損。 |
| **謊稱「已送醫審會鑑定」** | 檢察官職務上之誠實義務 | 造成請求權人空等10個月，錯失黃金救濟期間，構成怠於執行職務及不實陳述。 |
| **刪除法醫報告關鍵但書，逕行認定「自然死」** | 檢察官職務上之客觀義務 | 偽造文書或不實記載，以虛偽死因結案，剝奪真相調查機會，嚴重侵害請求權人知悉真相之權利。 |
| **未記載林怡婷陳述於筆錄、未讓其簽名確認** | 偵查筆錄製作之程序正義 | 系統性刪除證人陳述，架空程序主體地位，構成程序不法之核心事實。 |

### 3.2 法院機關之程序不法

#### 3.2.1 臺灣臺中地方法院（法官巫淑芳、孫藝娜）

| 違失行為 | 法律依據/義務 | 法律意涵與攻防重點 |
| :--- | :--- | :--- |
| **駁回證據保全聲請（114聲424）** | 《民事訴訟法》第368條（證據保全） | 以「已有紙本病歷」為由駁回，未適用後段「確定現狀之法律上利益」，導致關鍵電子病歷證據無法保全，核心問題無法進入審查。此為程序上對證據權之侵害。 |
| **駁回訴訟救助（114救223）** | 《民事訴訟法》第109條（釋明無資力）、最高法院107台抗863裁定意旨 | 認定中低收入戶證明不足釋明無資力，且以法扶駁回為由推論「非無資力」，構成論理矛盾及循環論證，違背最高法院意旨，剝奪弱勢當事人接近法院之權利。 |
| **同日命補繳高額裁判費且不得抗告（114補2969）** | 《民事訴訟法》第109條之1（訴訟救助裁定確定前不得駁回本訴） | 訴訟救助駁回同日即命補繳，且限制抗告權，嚴重壓縮救濟空間，涉嫌剝奪請求權人程序救濟機會。 |
| **以未繳費為由駁回本訴（115醫5）** | 《民事訴訟法》第249條第1項第6款 | 民事賠償訴訟因程序門檻而遭終結，構成對訴訟權的實質侵害。 |

#### 3.2.2 臺灣高等法院臺中分院（審判長許秀芬、法官吳國聖、陳佩怡）

| 違失行為 | 法律依據/義務 | 法律意涵與攻防重點 |
| :--- | :--- | :--- |
| **將「釋明」提高至「證明」程度** | 最高法院99台抗853（釋明與證明之區別） | 違法提高釋明門檻，加重請求權人舉證責任，阻礙程序進行。 |
| **引第13條證明證據存在，又以「未釋明」駁回** | 《民事訴訟法》第469條第6款（裁定理由矛盾） | 裁定理由自相矛盾，構成當然違背法令，實質上封鎖了請求權人的程序救濟途徑。 |

## 4. 「程序不法」作為國家賠償訴訟核心攻防策略

### 4.1 構成要件分析

*   **公務員執行職務行使公權力**：上述所有違失行為均發生於檢察官、法官及法扶人員執行公務期間。
*   **故意或過失**：檢察官的謊稱、刪除筆錄，法院的論理矛盾、違法駁回，法扶的不當駁回，均可主張其具有故意或重大過失。
*   **不法侵害人民自由或權利**：這些程序不法行為共同導致請求權人憲法第16條所保障之「訴訟權」被侵害，包括接近法院權、受合法程序保障權、證據權、聽審權等。
*   **損害發生**：訴訟權被侵害直接導致請求權人無法查明真相、無法獲得公正審判、承受巨大精神痛苦，以及因程序延宕而產生的財產損失。

### 4.2 策略應用重點

1.  **強調「系統性排除」**：將各機關單一的程序違失串聯起來，論述其共同構成一個「系統性排除弱勢當事人接近法院」的結構性障礙，而非單一偶發事件。這將加重國家賠償的責任基礎。
2.  **聚焦「訴訟權侵害」**：明確指出這些程序不法行為直接侵害了請求權人的訴訟權，而非僅是程序瑕疵。訴訟權是憲法保障的基本權利，其侵害應受國家賠償。
3.  **利用律師評析強化論證**：將律師評析中對各項程序違失的法律分析（如傳喚程序無效、訴訟救助裁定違背最高法院意旨、證據保全裁定理由矛盾等）作為強有力的論證依據，證明公務員行為的違法性。
4.  **連結損害賠償**：將程序不法造成的精神痛苦（兩年半訴訟折磨、真相剝奪）、訴訟權侵害（無法有效接近法院）、程序不公等非財產上損害，與公務員的違失行為直接連結，主張高額精神慰撫金。
5.  **證據為王**：再次強調所有相關法律文件（裁定書、不起訴處分書、法扶駁回函等）均為證明這些程序不法的直接證據。

## 5. 實務執行建議

1.  **協議階段準備**：在與賠償義務機關協議時，應強硬地提出這些「程序不法」的具體事實與法律依據，明確指出其行為已構成國家賠償責任，並強調若協議不成，將在訴訟中全面揭露這些程序違失。
2.  **訴訟階段攻防**：若進入訴訟，應在起訴狀中詳細闡述上述各項程序不法，並引用律師評析中的法律見解，作為主張公務員過失或故意侵害訴訟權的關鍵論點。特別是針對地院同日命補費且不得抗告、高院裁定理由矛盾等點，應作為程序不法的重點攻防。
3.  **證據保全之損害認定**：高院裁定雖引醫療法規確認證據應存在，卻以未釋明毀損風險駁回保全聲請，此種理由矛盾依《民事訴訟法》第469條第6款屬當然違背法令，可作為主張公務員過失之依據，並連結至真相剝奪的損害。

## 6. 結論

「程序不法」是本案國家賠償訴訟的強力武器。透過系統性地揭露檢察、審判及法律扶助機關在程序上的種種違失，並將其上升至侵害憲法訴訟權的高度，將能有效強化國家賠償請求的正當性與說服力，爭取請求權人應得之賠償。
